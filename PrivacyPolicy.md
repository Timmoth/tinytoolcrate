# Privacy Policy for tinytoolcrate

**Effective Date:** April 2026

tinytoolcrate is designed to operate entirely within the user’s browser and does not collect, store, or transmit any personal data.

## Data Collection

tinytoolcrate does **not collect, store, or share** any personally identifiable information or browsing data.

## How the Extension Works

When the user activates the extension, it processes the content of the currently active webpage locally in the browser to generate a clean markdown version. This processing does not involve sending data to any external servers.

## Permissions Usage

* **activeTab / host permissions**: Used to access the current webpage only when the user activates the extension.
* **scripting**: Used to extract and format page content.
* **clipboardWrite**: Used only when the user chooses to copy output.
* **downloads**: Used only when the user chooses to download the generated markdown file.

## Data Sharing

No data is transmitted, shared, or sold to any third parties.

## Remote Code

All code runs locally within the extension. No remote code is executed.

## Changes

This privacy policy may be updated if the extension’s functionality changes.

## Contact

If you have any questions, please contact the developer via the Chrome Web Store listing.
